Lottie awesome

Lottie awesome is a set of icons and scripts for controlling AfterEffects bodymovin (also called lottie) animations.

How to use?

Clone the repository, or download the zip file from https://github.com/torabian/lottie-awesome, and you can access the fonts folder all json file you can see here, then import it using your lottie environment
